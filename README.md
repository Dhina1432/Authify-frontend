# Authify-frontend 

Authify is a modern authentication web application built with **React** and **Vite**. It provides a clean and responsive user interface for user registration, login, password reset using OTP, and account verification.

## 🌐 Live Demo

https://authify-frontend-x3nq.vercel.app

## 📸 Features

- 🔐 User Registration
- 🔑 User Login
- 🚪 Secure Logout
- 📧 Forgot Password with OTP
- 🔄 Reset Password
- ✅ Email Verification
- 🍪 JWT Authentication using HTTP-Only Cookies
- 📱 Responsive UI
- ⚡ Fast performance with Vite

## 🛠️ Tech Stack

- React 19
- Vite
- React Router DOM
- Axios
- Bootstrap 5
- Bootstrap Icons
- React Toastify

## ⚙️ Environment Variables

Create a `.env` file in the root directory.

```env
VITE_BACKEND_URL=http://localhost:8080/api
```

For production:

```env
VITE_BACKEND_URL=https://your-backend-url/api
```

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Dhina1432/Authify-frontend.git
```

Navigate into the project

```bash
cd Authify-frontend
```

Install dependencies

```bash
npm install
```

Run the application

```bash
npm run dev
```

Build for production

```bash
npm run build
```
